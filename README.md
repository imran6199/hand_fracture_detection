# Hand Fracture Detection App

A web application that detects hand fractures from X-ray images using a YOLOv8 object detection model. Built with **Streamlit** for a simple web interface and **Ultralytics YOLOv8** for AI-powered fracture detection.

---

## Features

- Upload X-ray images of hands.
- Detect fractures and annotate them in the image.
- Display the annotated image directly in the web app.
- Lightweight and fast inference using YOLOv8.
